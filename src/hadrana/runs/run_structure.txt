Filename of HDF5 file per fit comprised of

<ensmemble_id>
<observable>
<nsquare0X>
<fit_range_min>
<fit_range_max>
<model_id>
<correlation_type>

RUN_ROOT = Path("/home/ck/phd/results/c2pt/D251")
RUN_ID = "run01"

RUN_PATH = RUN_ROOT / RUN_ID

there is a run overview file specifying the purpose of each fit run
D251_c2pt_run_overview.txt
